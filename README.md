# react-ecommerce-ui 

A react project created using Shopify rest apis

### Framework Stack

- React
- Router
- Hooks
- Thunk
- Redux
- JEST

## Running Locally


```sh
$ gh repo clone resobyte/react-ecommerce-ui
$ cd react-ecommerce-ui
$ npm install
$ npm start
```
